# 组件库架构搭建
  ## 使用 pnpm 初始化项目
  * 安装 `pnpm`（npm install -g pnpm），执行 `pnpm init` 生成 package.json 
  * 新建 pnpm-workspace.yaml 文件，里面配置工作目录

    ``` javascript
    packages:
      - 'packages/\*'
    ```
  * 新建 .gitignore 文件，配置:
    ``` javascript
    node_modules
    ```
    

  ## 定义开发规范
  采用 eslint 做代码检查规范，这里 -w 的意思是需要在根项目，也就是在最外层进行安装才行，不然会报错
  ``` js
  pnpm i eslint -D -w
  npm eslint --init
  ```
